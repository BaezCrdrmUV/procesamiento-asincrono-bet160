Ejemplo realizado en C# con el framework .net Core 3.1

DESCRIPCIÓN DEL EJEMPLO

En este ejemplo se realizaron 2 métodos, uno muestra un mensaje y el otro calcula el 50% de una cantidad ingresada por el usuario. Cuando se ejecuta el programa primero se pide al usuario que ingrese una cantidad de la cual se calculará el porcentaje, después se hace una llamada al método asíncrono que se encargara del cálculo, así como del método que emitirá un mensaje. Suponiendo que el usuario ingreso la cantidad 70, el resultado debe ser el siguiente:

Ingrese una cantidad entera para calcular su 50%
70
Tarea de calcular el 50% de una cantidad ejecutandose de manera asincrona
Ejecucion asincrona del metodo 1
Tarea terminada. El 50% de 70 es: 35
